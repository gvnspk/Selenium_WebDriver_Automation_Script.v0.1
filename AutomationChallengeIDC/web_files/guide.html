Introduction

Please read the instructions carefully

This page was made to test the understanding and capabilities of potential automation engineer candidates with good programming skills.

Using the selenium based framework (or wrapper) of your choice, write some sample tests to complete the objectives outlined in the tests below (Test 1 to Test 4) using properly formatted selectors to interact with the page's elements.

Complete Test 5 and Test 6 using programming language of your choice (preferably Scala/Java) following programming standards and comments

Note: Use the latest Chrome version for automating following test cases.

Test 1

  • Navigate to the home page
  • In Test 1 div click the "Open Modal" Button
  • Enter values for Name and City
  • Click the "Enter Data" Button
  • Close the Modal
  • Verify values under Open Modal button that were entered in the textboxes

Test 2

  • Navigate to the home page
  • In Test 2 div click on "Click Me"
  • Verify and print all the available options
  • Search for the keyword "Angular"
  • Verify two options are visible as "Angular" and "Angular 2"
  • Clear the search box and type "ReactJs"
  • Verify no option is visible

Test 3

  • Navigate to the home page
  • In Test 3 div search box enter value "USA"
  • Verify table displays two rows with country as USA
  • Clear search box
  • Verify table displays four rows

Test 4

  • Navigate to the home page
  • In Test 4 div drag button "Drag Me" to the Red Rectangle box
  • After dragging, verify if button was dragged inside the box

Test 5

  • Write a program that prints the numbers from 1 to 100. But for multiples of three print "MThree" instead of the number and for the multiples of five print "MFive". For numbers which are multiples of both three and five print "Resolver".
  • Requirements:
  • Create a function
  • Use loops
  • Use conditions
  • Must be typesafe
  • In your program, call the function that you created
  • Include comments where necessary

Test 6

  • Write a program that defines a shape class with a constructor that gives value to width and height.
  • Define two sub-classes triangle and rectangle that calculates the area of the shape area()
  • In the main, define two variables a triangle and a rectangle and then call the area() function in this two varibles.
